= 5-1 원시값의 메서드
자바스크립트에서는 원시값String ,Number 등을 마치 객체처럼 다룰수 있게해준다.
 원시형 != 객체
 원시형: string,number,bigint,boolean,symbol,null,undefined 총 7개
 객체: 프로퍼티에 다양한 종류의 값을 저장 가능
 객체의 장점중 하나는 함수를 프로퍼티로 저장가능

let str = "Hello";
str.test = 5;
alert(str.test);
 원시값은 추가 데이터를 저장할수없다.


= 5-2 숫자형

모던 자바스크립트에서 숫자를 나타내는 자료형은 두가지
1.number:일반적인 숫자는 65비트형식의 IEEE-754에 저장된다.
2.BigInt: 임의의 길이를 가진 정수는 BigInt, 아주 특별한 경우에서만 사용됨

-숫자 입력하는 다양한법 

let billion=1e9			// 10억, 1과 9개의 0

let ms = 1e-6			// 1에서 왼쪽으로 6번 소수점으로이동

e우측에 음수가 있으면 이 음수의 절댓값만큼 10을 거듭제곱한 수로 나눔,  e왼쪽의 수(양수라면)에 e오른쪽에 있는 수만큼 거듭제곱함


-16진수,2진수,8진수

-16진수
alert(0xff)			// 16진수로 표현하는 방법으로 255
alert(0xFF)		// 대문자로해도 같은 값을 반환

let a= 0b11111111		// 255의 2진수
let b = 0o377		// 255의 8진수

alert(a==b);		//true 진법은 다르지만 a와 b는 같은수이기 때문이다.

이외의 진법을 사용하려면 parseInt를 사용해야한다.
